Works

Algomint

I worked on the backend exchange logic for Algomint mostly in Go but also some NodeJS. I was also in a DevOps role, setting up the infrastructure with Terraform, Kubernetes, Docker and Helm as well as doing production deployments. Pretty interesting the problem area of being able to use a given cryptocurrency on a foreign blockchain.

Meld Gold

I did some early backend/devops work for Meld Gold consisting of the migration from Digital Ocean to Google Cloud Platform, using Terraform, Kubernetes, Docker, Helm, NodeJS and Go. Had some fun mulling over the problem area of gold digitalization here.

Dragon Code

A hobby project meant for players of Ravenloft PotM, a NWN persistent world. It's hosted on github and is a tool that enables players to easily encrypt and decrypt text and use it in the game. The cipher is shifted variant of the combination of the One Time Pad and the Vigenere Cipher.

Amazon Fruit Festival

A live Wordpress site that I set up Dockerized on a VPS and maintained it for the Amazon Fruit Festival. There you can find information about the festival and buy tickets through PayPal or directly with Bitcoin (Without a middleman!).

Nextcloud Instance

I stayed at the Fruithaven Ecovillage a while back. Eventually I set up a Dockerized Nextcloud on a VPS for them with routine backups. I voluntarily maintained it.
